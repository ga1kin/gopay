### gopay - пример простейшего парсера.
#### Учебный проект для слушателей курса.

Получает график выплаты пенсии с сайта [НПФ "Благосостояние"](https://npfb.ru/grafik-vyplaty-pensii.php)

#### Пример работы:
**Запуск без аргумента:** 
```
Укажите команду для выполнения:
	
	Московский
	Приволжский
	Восточно-Сибирский
	Северный
	Горьковский
	Юго-Восточный
	Свердловский
	Южно-Уральский
	Северо-Кавказский
	Забайкальский
	Красноярский
	Западно-Сибирский
	Куйбышевский
	Дальневосточный
	Октябрьский
	Калининградское

```
**Запуск с ключом**
./gopay Северный

```Северный филиал 05.04.2023
```
